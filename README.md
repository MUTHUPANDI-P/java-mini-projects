# Java Mini Projects – Internship Task

This repository contains three Java mini projects developed as part of my internship.

## Projects Included

### 1. Library Item Management System
- Manages Books, Magazines, and DVDs
- Demonstrates OOP concepts:
  - Inheritance
  - Polymorphism
  - Access Modifiers

📂 Folder: `LibraryProject`

---

### 2. Student Course Registration System
- Manages student records
- Features:
  - Add and view students
  - Filter students with marks > 70
  - Group students by course
  - Sort students by marks

- Concepts Used:
  - Collections (List)
  - Stream API
  - Lambda Expressions
  - Exception Handling

📂 Folder: `StudentProject`

---

### 3. Multi-Threaded Log File Analyzer
- Reads a log file and processes it using multiple threads
- Counts occurrences of:
  - INFO
  - ERROR
  - WARNING

- Concepts Used:
  - Multithreading
  - File Handling

📂 Folder: `LogAnalyzerProject`

---

## Technologies Used
- Java
- Eclipse IDE

---

## How to Run

1. Clone or download the repository
2. Open project in Eclipse IDE
3. Run the main class:
   - Library → `LibraryApp.java`
   - Student → `StudentApp.java`
   - Log → `LogApp.java`

---

## Note
- Ensure `log.txt` is present in LogAnalyzerProject before running.
- All projects are console-based Java applications.
